# Personal Portfolio (Django)

A personal portfolio web application built with Django to showcase my projects, skills, and professional experience. This project was developed as a hands-on way to improve my backend development skills and gain more experience with Django.

## Overview

This application serves as a central hub for my professional information, including:

- Projects portfolio  
- Professional experience  
- Technical skills  
- Downloadable Curriculum Vitae (CV)  
- Social media links  

The main goal of this project was to practice Django development by building a simple yet functional and structured web application.

## Technologies Used

- Python  
- Django  
- HTML5 & CSS3   
- SQLite  

## Features

- Dynamic project listing  
- Organized experience section  
- Skills showcase  
- CV download functionality  
- Contact and social links integration

This project was created primarily for learning purposes, focusing on:

* Understanding Django architecture
* Working with models, views, and templates
